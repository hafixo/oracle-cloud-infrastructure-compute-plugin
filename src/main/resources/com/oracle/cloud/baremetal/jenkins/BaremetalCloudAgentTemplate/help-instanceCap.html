This functionality places a limit on the number of OCI Instances that Jenkins may launch from this Template.

For example, if this field is 3, Jenkins will only launch a maximum of 3 OCI Instances for this Template. Template Instance Cap cannot exceed the Cloud Instance Cap.
For example, if Cloud Instance Cap is 2, and the Template Instance Cap is 3, only 2 Instances can be created.

Leave this field empty to remove the Template Instance Cap.
